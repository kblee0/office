# StyleX RibbonX PowerPoint 매크로 사용 설명서

## 개요
이 프로젝트는 PowerPoint의 RibbonX 커스텀 툴바와 VBA 매크로를 통합한 스타일 도구 모음입니다.

## 파일 구조
- `sample_ribbonx.xml` - RibbonX 커스텀 UI 정의
- `StyleX.bas` - VBA 매크로 코드
- `SPEC.md` - 기능 명세서

## 설치 방법

### 1단계: PowerPoint PPTM 파일 생성
PowerPoint를 열고 매크로 사용 가능한 프레젠테이션을 생성합니다 (.pptm 형식).

### 2단계: VBA 매크로 코드 추가
1. PowerPoint에서 `Alt + F11`을 눌러 VBA 에디터를 엽니다
2. `삽입` > `모듈` 클릭
3. `StyleX.bas` 파일의 모든 코드를 복사하여 새 모듈에 붙여넣습니다
4. 파일을 저장합니다

### 3단계: RibbonX XML 연결
PowerPoint 파일의 구조를 수정하여 RibbonX를 적용합니다:

1. `.pptm` 파일을 ZIP 파일로 변환 (확장명을 .zip으로 변경)
2. `_rels` 폴더를 열고 `.rels` 파일 편집
3. 다음 줄을 추가:
   ```xml
   <Relationship Id="rId99" Type="http://schemas.microsoft.com/office/2007/relationships/ui/extensibility" Target="customUI/ribbonx.xml"/>
   ```

4. `customUI` 폴더 생성 (없으면)
5. `sample_ribbonx.xml`을 폴더 내에 `ribbonx.xml`로 저장
6. ZIP 파일을 다시 `.pptm` 확장명으로 변환

### 4단계: 이미지 파일 추가
각 색상칩에 대해 PNG 이미지 파일이 필요합니다:
- 이미지 크기: 16x16 또는 32x32 픽셀
- 이미지명: `C` + 16진수 색상값 (예: `C2F2F2F.png`)
- 저장 경로: `ppt/media/` 폴더

## 기능 설명

### 그룹1: 채우기 (배경색) - 다중 선택 지원
여러 도형을 동시에 선택하여 배경색을 변경할 수 있습니다.
- 도형 또는 셀 선택 후 원하는 색상 클릭
- 표의 경우: 선택된 셀만 배경색 변경
- 12가지 사전정의 색상 사용 가능

### 그룹2: 선색 (테두리선) - 다중 선택 지원
여러 도형을 동시에 선택하여 테두리선 색상을 변경할 수 있습니다.
- 도형 또는 셀 선택 후 원하는 색상 클릭
- 표의 경우: 선택된 셀의 테두리 색상만 변경

### 그룹3: 사각형도구 - 다중 선택 지원
여러 도형을 동시에 선택하여 형태를 변경할 수 있습니다.
- 도형 선택 후 원하는 형태 클릭
- 선택된 모든 도형이 동시에 변환됩니다:

### 그룹4: 표 도구
선택된 표의 테두리선을 설정합니다:
- 0.5pt/1.0pt/1.5pt 모든 테두리
- 0.5pt/1.0pt/1.5pt 좌우 제외 (상하만)

### 그룹5: 폰트 도구
텍스트 폰트 속성을 한 번에 변경합니다 (폰트명, 색상, 굵은체):
- 본고딕 검정색 굵은체
- 맑은고딕 라이트 블루 일반
- 한컴산체 빨간색 굵은체
- 궁서 파란색 굵은체
- 돋움 초록색 일반

## 사용 팁

1. **다중 선택**: 그룹1~3은 Ctrl+클릭으로 여러 도형을 선택하여 동시에 적용 가능
2. **색상 도구**: 색상 도구 사용 시 도형을 먼저 선택해야 합니다
3. **표 선택**: 표 도구 사용 시 표 내 셀을 선택하고 사용합니다
4. **텍스트 선택**: 폰트 도구 사용 시 텍스트를 포함한 도형을 선택합니다
5. **Ctrl+Z로 취소**: 모든 작업을 실행 취소할 수 있습니다

## 오류 처리

매크로 실행 중 다음과 같은 오류가 발생할 수 있습니다:

- "선택된 도형이 없습니다" - 먼저 도형을 선택한 후 시도
- "선택된 표가 없습니다" - 표를 선택한 후 시도
- "텍스트를 포함한 도형을 선택해주세요" - 텍스트가 있는 도형 선택

## 커스터마이징

### 색상 추가/변경
1. `sample_ribbonx.xml`에서 새 버튼 추가
2. `ID`와 `image` 속성을 새 색상 코드로 설정
3. 대응하는 PNG 이미지 생성 및 저장

### 폰트 도구 커스터마이징
1. `sample_ribbonx.xml`에서 버튼의 `tag` 속성 수정
   형식: "폰트명,색상16진수,굵은체여부"
   예: "Arial,FF0000,true"
