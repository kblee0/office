# 기능
- 개요 : RibbonX PowerPoint 툴바
- 색상칩
  - 이미지 png icon을 12개를 넣고 image id 는 'C' + 색상 16진수 형태로 구성 (예: C2F2F2F)
- 그룹1: 채우기 (선택된 모든 쉐이프 또는 셀의 경우 선택된 셀만)
  - 12개의 색상칩 버튼 구성
  - 색상칩 클릭시 선택한 도형 또는 그래픽 표 또는 셀의 배경색 변경
  - 색상은 image id값 (예: C2F2F2F에서 오른쪽 6자리 16진수 이용)
- 그룹2: 선색 (선택된 모든 쉐이프 또는 셀의 경우 선택된 셀만)
  - 12개의 색상칩 버튼 구성
  - 색상칩 클릭시 선택한 도형 또는 그래픽 표 또는 셀의 테두리선 색상 변경
- 그룹3: 사각형도구 (선택된 모든 도형)
  - 버튼1: 도형을 직사각형으로 변경
  - 버튼2: 도형을 0.1cm 둥근모서리 직사각형으로 변경
  - 버튼3: 도형을 0.2cm 둥근모서리 직사각형으로 변경
  - 버튼4: 도형을 0.1cm 위쪽 둥근모서리 직사각형으로 변경 (아랫쪽은 각진사각형)
  - 버튼5: 도형을 0.2cm 위쪽 둥근모서리 직사각형으로 변경 (아랫쪽은 각진사각형)
  - 버튼6: 도형을 0.1cm 아래쪽 둥근모서리 직사각형으로 변경 (윗쪽은 각진사각형)
  - 버튼7: 도형을 0.2cm 아래쪽 둥근모서리 직사각형으로 변경 (윗쪽은 각진사각형)
- 그룹4: 표 도구 (표는 하나만 선택 가능)
  - 버튼1: 표의 모든 테두리선을 0.5pt 직선으로 변경
  - 버튼1: 표의 모든 테두리선을 1.0pt 직선으로 변경
  - 버튼1: 표의 모든 테두리선을 1.5pt 직선으로 변경
  - 버튼1: 표의 모든 테두리선을 0.5pt 직선으로 변경하고 표전체의 좌우 테두리를 지움
  - 버튼1: 표의 모든 테두리선을 1.0pt 직선으로 변경하고 표전체의 좌우 테두리를 지움
  - 버튼1: 표의 모든 테두리선을 1.5pt 직선으로 변경하고 표전체의 좌우 테두리를 지움
- 그룹5: 폰트 도구 (단, 버튼의 tag를 "폰트명,색상16진수,굵은체여부"를 설정)
  - 버튼1~5: tag값으로 선택된 텍스트 또는 도형 또는 표 또는 셀의 텍스트를 tag에 설정된 폰트속성으로 변경
